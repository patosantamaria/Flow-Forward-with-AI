## 1. Vision & Purpose

Our vision is to embed Artificial Intelligence into the core of our operations, workflow and decision-making processes, transforming [Business Unit] into a more efficient, innovative, and data-driven organization.

The purpose of this journey is to empower our colleagues by:

- Equipping them with practical skills to leverage Gemini and other basic AI tools to solve real-world business challenges.
- Exciting colleagues about the possibilities Gen AI can offer them instead of seeing it as a threat or something too complex.

---

## 2. The Modular Journey

This program is structured into 11 self-contained modules to allow for flexible delivery (Full-Day, Half-Day, or Remote "Quick Hits").

| Phase                    | Core Focus           | Key Outcome                                                           |
| :----------------------- | :------------------- | :-------------------------------------------------------------------- |
| **Foundation** (M1–M5)   | Foundations & Ethics | Shared vocabulary, Zero Trust security, and TCF prompting skills.     |
| **Toolbox** (M6–M9)      | Core Tools Deep Dive | Proficiency in Workspace Side Panel, Web App, Mobile, and NotebookLM. |
| **Automation** (M10–M11) | Automate & Build     | Ability to build custom Gems and no-code Studio automations.          |

## 3. Scope

| In-Scope                                                                               | Out-of-Scope                                                |
| :------------------------------------------------------------------------------------- | :---------------------------------------------------------- |
| [Business Unit] staff across all functions (Operations, HR, Finance, Commercial, etc.) | Building custom AI models or major infrastructure overhauls |
| Empowering colleagues through skills development, training, and cultural change        | Procurement of new, large-scale AI platforms                |
| Leveraging existing AI tools like Gemini in Workspace, NotebookLM, and Gems            | Complex data science projects led by specialized teams      |
| Identification, prototyping, and sharing of business-relevant use cases                |                                                             |
| **Global BU reuse** — curriculum repackaged for any Vopak BU worldwide                 |                                                             |

---

## 4. Success Metrics

### Adoption Metrics

- Number of active weekly Gemini users.
- Usage growth rate per function.

### Impact Metrics

- Number of AI-enabled workflows automated.
- Estimated hours saved per month.
- Measurable impact on productivity, safety, and commercial effectiveness.

---

## 5. Resourcing

- Time allocation for local AI Champions to facilitate and support.
- Time allocation by participants to follow the program modules.
- Global reuse of curriculum materials incurs no additional content licensing cost.
