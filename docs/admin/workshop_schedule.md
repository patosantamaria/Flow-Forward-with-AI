# AI Workshop Schedules (Modular)

> **Facilitator Note:** These schedules are templates based on the [Module Catalogue](../modules/module_catalogue.md). All times show **duration**, not fixed start times — adapt to your local session schedule. Choose the configuration that fits your available time and audience.

---

## 🌅 Option 1: Full-Day "AI Week" Summit (~7.5 hrs total)

**Modules included:** M1–M9 (Foundation + All Core Tools)

| Duration | Module(s) | Facilitator Action                                                          |
| :------- | :-------- | :-------------------------------------------------------------------------- |
| 30 min   | M1 + M2   | **Intro:** Vision, Vopak career superpower framing, what GenAI actually is. |
| 30 min   | M3 + M4   | **Guardrails:** Security, Zero Trust, No PII. Intro TCF Framework.          |
| 15 min   | ☕ Break  |                                                                             |
| 60 min   | M6        | **Side Panel:** Gmail, Docs, Drive. Cross-app context. Guided practice.     |
| 60 min   | M7        | **Web App:** Heavy lifting, file uploads, analysis. "Time Saved" exercise.  |
| 60 min   | 🍽️ Lunch  |                                                                             |
| 60 min   | M9        | **NotebookLM:** The Vault. Audio overviews. Knowledge base building.        |
| 30 min   | M8 + M5   | **Mobile & Demo:** Field worker scenarios. Live "The Difference" demo scan. |
| 60 min   | Practice  | **Sprint:** Solve a real weekly team task using any of the day's tools.     |
| 30 min   | Showcase  | Teams present their "Time Saved" result or NotebookLM discovery.            |
| 15 min   | Wrap-Up   | Survey & induction into the [CoE Registry](../phases/post_training_coe.md). |

## 🛠️ Option 3: Automate & Build (~4 hrs total)

**Modules included:** M10 + M11

| Duration | Module   | Facilitator Action                                                   |
| :------- | :------- | :------------------------------------------------------------------- |
| 120 min  | M10      | **Gems Workshop:** Build your AI colleague. (Guided + Solo build).   |
| 15 min   | ☕ Break |                                                                      |
| 105 min  | M11      | **Automation:** Workspace Studio (Flows) + Apps Script (The Bridge). |

> **⏰ Manager Commitment Required:** M10 and M11 are the most hands-on and cognitively demanding modules in the programme. Participants need **uninterrupted focus time** — not a session squeezed between other meetings. Managers must actively block this time in participants' calendars and communicate that this is protected learning time, not a voluntary add-on. If scheduling is tight, prioritize M10 (Gems) as the minimum — it delivers the highest individual ROI.
