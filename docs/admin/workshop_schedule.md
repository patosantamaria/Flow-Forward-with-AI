# AI Workshop Schedules (Modular)

> **Facilitator Note:** These schedules are templates based on the [Module Catalogue](../modules/module_catalogue.md). Choose the one that fits your session format.

---

## 🌅 Option 1: Full-Day "AI Week" Summit (09:00 – 16:30)

**Modules included:** M1–M9 (Foundation + All Core Tools)

| Time              | Module(s) | Facilitator Action                                                          |
| :---------------- | :-------- | :-------------------------------------------------------------------------- |
| **09:00 – 09:30** | M1 + M2   | **Intro:** Vision, Vopak career superpower framing, what GenAI actually is. |
| **09:30 – 10:00** | M3 + M4   | **Guardrails:** Security, Zero Trust, No PII. Intro TCF Framework.          |
| **10:00 – 10:15** | Break     |                                                                             |
| **10:15 – 11:15** | M6        | **Side Panel:** Gmail, Docs, Drive. Cross-app context. Guided practice.     |
| **11:15 – 12:15** | M7        | **Web App:** Heavy lifting, file uploads, analysis. "Time Saved" exercise.  |
| **12:15 – 13:15** | LUNCH     |                                                                             |
| **13:15 – 14:15** | M9        | **NotebookLM:** The Vault. Audio overviews. Knowledge base building.        |
| **14:15 – 14:45** | M8 + M5   | **Mobile & Demo:** Field worker scenarios. Live "The Difference" demo scan. |
| **14:45 – 15:45** | Practice  | **Sprint:** Solve a real weekly team task using any of the day's tools.     |
| **15:45 – 16:15** | Showcase  | Teams present their "Time Saved" result or NotebookLM discovery.            |
| **16:15 – 16:30** | Wrap-Up   | Survey & induction into the [CoE Registry](../phases/post_training_coe.md). |
