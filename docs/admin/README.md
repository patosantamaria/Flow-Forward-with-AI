# Admin & Guides Directory

This directory serves as the control centre for the "Flow Forward with AI" programme.

It contains all the materials required to launch, monitor, and formally facilitate the training. Unlike the `modules/` or `phases/`, these files are not meant for the average participant. They are designed for program managers, AI Champions, and workshop facilitators.

## Contents:

- **Participant Trackers:** Schema and guides for monitoring who attended and tracking badge progression.
- **Surveys:** Baseline, Check-In, and Final Feedback forms to measure confidence and ROI.
- **Email Templates:** Standardized outward communication for invites and "Tip of the Week" follow-ups.
- **Master Facilitator Guides:** Centralised handbooks instructing trainers on how to handle resistance (`resistance_handling.md`), route participants through the matrix (`_department_guide.md`), and execute high-pressure simulations (`capstone_facilitator_guide.md`).
- **Project Charter:** The foundational document outlining the goals and the "Why" behind the CoE's investment in Gemini.
