## 1. Product Overview — Hexamethylene Diamine (HMD)

| Property             | Value                                     |
| :------------------- | :---------------------------------------- |
| Chemical Name        | Hexamethylene Diamine (1,6-Diaminohexane) |
| CAS Number           | 124-09-4                                  |
| UN Number            | UN 2280                                   |
| Hazard Class         | Class 8 — Corrosive                       |
| Subsidiary Risk      | Class 6.1 — Toxic                         |
| Physical State       | Solid at ambient; liquid above 42°C       |
| Flash Point          | 71°C (closed cup)                         |
| Melting Point        | 42°C                                      |
| Handling Temperature | 60-80°C (maintained in liquid state)      |

> ⚠️ HMD must be maintained above its melting point (42°C) during storage and transfer. Trace heating and temperature monitoring are critical.

## 2. PPE Requirements

### Minimum PPE for HMD Operations

| PPE Item               | Specification                                      | Certification       |
| :--------------------- | :------------------------------------------------- | :------------------ |
| Face Shield            | Full-face, chemical-splash rated                   | EN 166 / ANSI Z87.1 |
| Respiratory Protection | Full-face with organic vapour + acid gas cartridge | EN 14387 A2B2       |
| Chemical Suit          | Category III, Type 3 liquid-tight                  | EN 14605            |
| Gloves                 | Nitrile or butyl rubber, 0.4mm minimum thickness   | EN 374-3            |
| Safety Boots           | Chemical-resistant, steel toe                      | EN ISO 20345 S5     |

### PPE Certification Levels

| Level       | Training Duration | Covers                                                                   |
| :---------- | :---------------- | :----------------------------------------------------------------------- |
| Level 1     | 1 day             | Standard industrial PPE (hard hat, safety glasses, steel toes)           |
| Level 2     | 3 days            | Chemical-specific PPE (respirators, chemical gloves)                     |
| **Level 3** | **2 weeks**       | Full chemical suit, SCBA, decontamination procedures, emergency response |

> ⚠️ **All HMD operators require Level 3 PPE certification.** This takes a minimum of 2 weeks to obtain from an MOM-accredited training provider. Budget this into onboarding timelines.

## 3. Safety Induction Requirements

All new terminal operators must complete the following before first independent shift:

| Step | Activity                                             | Duration             | Responsible          |
| :--- | :--------------------------------------------------- | :------------------- | :------------------- |
| 1    | Vopak Fundamentals on Safety (e-learning + workshop) | Day 1                | SHEQ Manager         |
| 2    | Site-specific safety orientation (Sakra terminal)    | Day 1-2              | Terminal Manager     |
| 3    | IS-Safe tablet training                              | Day 2-3              | IT / SHEQ            |
| 4    | Product-specific chemical handling (HMD module)      | Day 3-4              | SHEQ Officer         |
| 5    | PPE Level 3 certification                            | 2 weeks              | External provider    |
| 6    | Buddy shadowing: minimum 10 supervised shifts        | 10 shifts            | Shift Superintendent |
| 7    | Competency assessment and sign-off                   | Day after 10th shift | Terminal Manager     |

**Total estimated onboarding time: 5-6 weeks** (including PPE Level 3 certification)

## 4. Permit to Work (PtW) Requirements for HMD

### Operations Requiring a PtW

- Opening any part of the HMD transfer system (lines, valves, Marine Loading Arms)
- Hot work within 15m of any HMD-containing equipment
- Entry to HMD tank bund area during transfer operations
- Any maintenance on trace heating systems

### Additional Safety Requirements

- Continuous **LEL (Lower Explosive Limit) monitoring** during all HMD transfer operations
- Standard alarm threshold: **20% LEL**
- Emergency isolation procedure must be rehearsed by every operator before first independent shift
- **Decontamination shower** must be tested and confirmed operational before any HMD handling

## 5. Emergency Response — HMD-Specific

### Exposure Response

| Exposure Type | Immediate Action                                                                        |
| :------------ | :-------------------------------------------------------------------------------------- |
| Skin contact  | Remove contaminated clothing. Flush with water for 20+ minutes. Seek medical attention. |
| Eye contact   | Flush with water for 20+ minutes. **Do not rub.** Immediate medical attention.          |
| Inhalation    | Move to fresh air. If breathing difficulty, administer oxygen. Call emergency services. |
| Ingestion     | Do NOT induce vomiting. Give water to rinse mouth. Immediate medical attention.         |

### Spill Response

1. Evacuate area (minimum 25m radius)
2. Don Level 3 PPE
3. Contain spill with chemical-resistant absorbent
4. **Do NOT wash into drains** — HMD is toxic to aquatic life
5. Notify Terminal Manager and SHEQ immediately
6. Report to NEA (National Environment Agency) if spill exceeds 50 litres
