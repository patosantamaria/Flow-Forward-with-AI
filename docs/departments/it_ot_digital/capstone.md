# IT, OT & Digital — "The Integration Blueprint" (Phase 3 Capstone)


---

## The Scenario

The CEO's AI mandate requires a technical roadmap for safe, governed AI adoption. The IT/OT/Digital team must assess the current infrastructure, define security guardrails, and propose an integration plan that enables all other departments to use AI effectively — without compromising operational technology security.

---

## Your Mission (Search → Apply → Refine)

### 1. Search (15 min)

> **Prompt:** "Research the current best practices for deploying Large Language Models in industrial environments with OT (Operational Technology) systems. Identify: (1) security risks unique to IT/OT convergence, (2) data isolation requirements, (3) authentication and access control frameworks, (4) regulatory considerations for industrial AI deployments."

### 2. Apply (30 min)

> **Prompt:** "Draft a technical roadmap for Vopak AME BU's AI integration. Structure as 3 phases over 12 months: Phase 1 — Security & Governance (SSO, data residency, acceptable use controls), Phase 2 — Pilot Deployment (Gemini + NotebookLM for 3 departments), Phase 3 — Scale & Monitor (telemetry, usage analytics via BigQuery, expansion). For each phase: objectives, deliverables, resources needed, risk mitigations."

> **Prompt:** "Draft an AI Governance Framework for Vopak. Cover: (1) data classification for AI inputs (what can/cannot be shared with AI tools), (2) model selection criteria, (3) output validation requirements, (4) incident response for AI-related data breaches, (5) audit trail requirements."

### 3. Refine (20 min)

Polish both documents into board-ready quality.

### 4. Optional Stretch Goal

Create a detailed architecture diagram (described in text) showing the data flow between Workspace, Gemini, NotebookLM, BigQuery, and terminal OT systems — with security boundaries clearly marked.
