# Departments Directory

This folder contains the tailored training matrices for the 14 Vopak departments participating in the Flow Forward with AI programme.

Each department folder follows a standardized structure:

- **`prompts.md`**: Department-specific use cases and PTCF prompts for Phase 1 & 2.
- **`gems.md`**: Instructions for the custom "Vopak Slide Designer" (Phase 2 Capstone) and an advanced role-playing "Agent" Gem (Phase 3).
- **`capstone.md`**: The Phase 2 Capstone Challenge scenario and instructions.
- **`capstone_facilitator_gem.md`**: The system instructions for the Facilitator Gem used to guide participants and inject the simulated cross-department update constraint.

> **Note:** For guidance on how to facilitate the capstone challenges and leverage NotebookLM synergy across multiple departments, refer to `docs/admin/_department_guide.md` and `docs/admin/capstone_facilitator_guide.md`.
