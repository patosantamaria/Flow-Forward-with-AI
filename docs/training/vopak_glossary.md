# Vopak Terminology Glossary — Prompt Grounding Reference

> **Purpose:** Use this glossary to "ground" your AI prompts with precise Vopak terminology. Adding a **Glossary Block** to your prompt prevents the AI from hallucinating or misunderstanding domain-specific terms.
>
> **How to use:** When crafting a prompt about a Vopak-specific topic, add: _"Use the following terms exactly as defined: [paste relevant terms from below]."_

## Operational, Safety & Product

| Term                  | Definition at Vopak                                                                                        |
| :-------------------- | :--------------------------------------------------------------------------------------------------------- |
| **ATG**               | Automatic Tank Gauging — primary system for product quantity measurement via tank level differentials.     |
| **IS-Safe**           | Intrinsically safe electronic tablets used by terminal operators in explosive field zones.                 |
| **TAS**               | Terminal Automation Blueprint — standard for terminal automation system architecture.                      |
| **PtW**               | Permit to Work — mandatory authorization for hazardous activities in Zone 0/1/2 environments.              |
| **LMRA**              | Last Minute Risk Assessment — mandatory safety check immediately before executing a PtW.                   |
| **PSSR**              | Pre-Startup Safety Review — mandatory SHEQ checklist during Cold Commissioning.                            |
| **THA**               | Terminal Health Assessment — triennial global safety audit program.                                        |
| **SOR**               | Safety Observation Round — a leading safety indicator tracked by SHEQ.                                     |
| **TIR**               | Total Injury Rate — a lagging safety indicator.                                                            |
| **PSER**              | Process Safety Event Rate — critical safety performance metric.                                            |
| **LOPC**              | Loss of Primary Containment — a critical process safety event (e.g., the Europoort TIER 2 fuel oil spill). |
| **NMR**               | Near-Miss Report — documented in the Enablon system.                                                       |
| **HiPo**              | High Potential — classification for near-misses that could have resulted in serious injury/fatality.       |
| **IN10 / IN11**       | Incident description formats: IN10 = brief (1-2 lines), IN11 = detailed (10 lines with root cause).        |
| **CRSA**              | Control Risk Self-Assessment — audit tool managed via Protiviti's Governance Portal.                       |
| **RBI**               | Risk Based Inspection — methodology governing maintenance inspection intervals.                            |
| **3YMP / 5YMP**       | 3-Year or 5-Year Maintenance Plan — long-term asset integrity planning documents.                          |
| **PDCA**              | Plan-Do-Check-Act — continuous improvement cycle for maintenance work order closure.                       |
| **MOC**               | Management of Change — safety process managed via Enablon, often cited as an admin bottleneck.             |
| **PHA**               | Process Hazards Analysis — required during MOC execution.                                                  |
| **ISGOTT**            | International Safety Guide for Oil Tankers and Terminals — mandatory compliance checklist.                 |
| **LOHC**              | Liquid Organic Hydrogen Carrier — key product in the New Energies strategy.                                |
| **SAF**               | Sustainable Aviation Fuel — emerging energy transition commodity.                                          |
| **HMD**               | Hexamethylenediamine — specialty chemical requiring strict temperature control (60°C ±2°C).                |
| **HAZOP**             | Hazard and Operability Study — systematic safety engineering analysis.                                     |
| **P&ID**              | Piping and Instrumentation Diagram — critical engineering documentation.                                   |
| **LEL**               | Lower Explosive Limit — safety measurement threshold (must be <10% before work in Zone 1).                 |
| **WBS**               | Work Breakdown Structure — financial decomposition used for CAPEX variance tracking.                       |
| **TIC**               | Total Installed Cost — forecasted total cost tracked via WBS variance analysis.                            |
| **API 650 / API 574** | American Petroleum Institute standards for tank construction and pipeline inspection.                      |
| **EEMUA 159**         | Engineering Equipment and Materials Users' Association standard for tank maintenance.                      |

## 💡 How to Use This in Your Prompts

**Without glossary grounding:**

> "Summarize this report about our tank operations."
> _(AI may misinterpret terms, produce generic output)_

**With glossary grounding:**

> "Act as a Vopak Terminal Operations Superintendent. Summarize this shift handover report. Use the following terms exactly: ATG = Automatic Tank Gauging, PtW = Permit to Work, IS-Safe = intrinsically safe tablets, LMRA = Last Minute Risk Assessment. Structure the summary by: Safety Incidents, Equipment Status (especially defeated safety-critical equipment), Active PtWs, Pending Maintenance."
> _(AI produces operationally precise, deployment-ready output)_
