# Phases Directory

This folder contains the high-level curriculums for the four primary phases of the "Flow Forward with AI" programme.

The phases define _what_ gets taught and in what sequence, whereas the actual atomic training materials (slides, prompts) live in `docs/modules/`.

- **`phase1_foundation_ethics.md`**: Core introduction, Zero Trust policies, and basic prompting (M1–M5). Designed as a standalone 90-minute session.
- **`phase2_core_tools.md`**: Deep dive into the four essential tools: Workspace, Web App, Mobile, and NotebookLM (M6–M9).
- **`phase1+2_combined.md`**: An aggregated half-day or full-day version that joins Phase 1 and Phase 2.
- **`phase3_automate_build.md`**: Advanced training on building AI colleagues with Gems and automating workflows with Apps Script (M10–M11).
- **`post_training_coe.md`**: The analytics and momentum loop (badges, champions, surveys) executed after a team completes the training.

**Phase 4** (Agents & Gemini Enterprise) is currently in development and operates at the organizational architecture level.
