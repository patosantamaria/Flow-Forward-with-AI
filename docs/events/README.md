# Events Directory

This folder contains the bespoke, highly-tailored training events that have been customised for specific Business Units (BUs) or specialized groups within Vopak.

Unlike the generic framework stored in `docs/departments/` and `docs/phases/`, the materials here are specifically built for real-world, scheduled interventions. They often contain:

- Precise running schedules and scripts for trainers
- Custom, synthetic "sandbox data" to safely execute challenges on
- Heavily modified slide prompts and Gem instructions

It is recommended to use the standard framework first, and only duplicate it into an `events/` sub-folder when a BU requests a massive departure from the baseline or requires unique sandbox data to be generated for their simulation.
