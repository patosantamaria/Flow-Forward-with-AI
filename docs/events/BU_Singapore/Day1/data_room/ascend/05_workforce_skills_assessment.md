# Workforce AI Skills Assessment Template

> **Purpose:** Baseline assessment to identify skill gaps across the organisation.
> **Instructions:** Copy into Google Sheets. Each row = one role. Use Gemini in Sheets to analyse gaps and generate training recommendations.

---

## Assessment Scale

| Level            | Description                                                              |
| :--------------- | :----------------------------------------------------------------------- |
| 1 — Unaware      | Has not used AI tools; limited understanding of what AI can do           |
| 2 — Aware        | Understands AI concepts; has seen demos but doesn't use AI in daily work |
| 3 — Practitioner | Uses AI tools (Gemini, NotebookLM) occasionally for specific tasks       |
| 4 — Proficient   | Uses AI daily; can write effective prompts; has built personal workflows |
| 5 — Champion     | Teaches others; builds Gems and automations; contributes to CoE          |

## Recommended Analysis Prompts
