## Notes

- CPF is calculated on **Ordinary Wages up to the ceiling of $6,800/month**
- If base + shift allowance + night premium exceeds $6,800, CPF is capped
- Budget should include a **10% contingency** for unexpected costs
- Rest-day work costs may increase if new MOM regulations take effect (check regulatory updates)
