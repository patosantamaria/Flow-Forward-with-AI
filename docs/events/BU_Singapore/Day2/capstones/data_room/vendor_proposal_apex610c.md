> ⚠️ **FACILITATOR NOTE — DO NOT SHARE WITH PARTICIPANTS:**
>
> **Red Flags the teams should find:**
>
> 1. **Carbon Steel casing + Carbon/Carbon seal faces** — Unacceptable for H₂S/Sour Crude service. Needs Duplex Stainless Steel casing and Tungsten Carbide / Silicon Carbide seal faces minimum.
> 2. **Nitrile (Buna-N) elastomers** — Will degrade rapidly in Sour Crude. Needs Viton/FKM or FFKM.
> 3. **Single Mechanical Seal** — Insufficient for toxic service (H₂S). Needs double/tandem seal with barrier fluid per API 682 Plan 53A.
> 4. **Non-Ex Rated motor** — The crude manifold area is classified Zone 1 IIC. A standard motor is an ignition risk. Needs Ex d IIC T3 minimum.
> 5. **"Partial Compliance" to API 610** — Originally built for clean water. This is a major concern for crude service.
> 6. **Smart-Link network access** — Requesting Ethernet connection to "nearest network switch" is a potential OT security breach. This should NEVER connect to the terminal control network.
> 7. **Cast Iron bearing housing** — Potentially unsuitable for the environment; corrosion risk.
