## Slide Notes

- **Strongly prefer a LIVE demo over pre-recorded screenshots.** If the internet is unreliable, prepare a backup screenshot set but always try live first.
- **Slide 4 Quick-Fire prompts** — customize per audience. Reference department prompt cards in `docs/departments/<department>/prompts.md` for ready-made examples.
- **Energy tip:** Ask the audience to raise their hand if they would actually send the bad prompt output unchanged. The silence is powerful.
- **If time is short:** Skip the quick-fire (Slide 4) and move directly to the M4 practice exercise.
