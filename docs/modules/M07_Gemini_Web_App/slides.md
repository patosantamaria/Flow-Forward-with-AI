## Slide Notes

- **Demo 1:** Use documents from your BU or a public example. If you don't have a real RFP, use two versions of the same SOP with deliberate conflicts pre-loaded.
- **Demo 2:** Always name a real competitor relevant to the audience (Oiltanking for Europe/Americas, Stolthaven for SEA). Vague "a competitor" produces weak demo results.
- **Slide 4 CRM warning:** Reinforce M3 zero trust — never paste live CRM/SalesForce data into the Web App. Public competitor data only.
- **Time Saved Exercise:** This is the emotional high point of M7. Give it the full allocated time. Never cut it for pace.
