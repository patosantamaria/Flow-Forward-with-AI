## Slide Notes

- **Slide 7:** Run this live — type the bad prompt first, show the result, then type the TCF version. Do NOT pre-prepare both — the live typing builds credibility.
- **Slide 8 (Practice):** Walk the room. If someone is stuck, ask "What's the most annoying part of your Monday morning?" — the answer becomes their context.
- **Most Common Mistake:** Participants mix up Task and Context. Context is background; Task is the action. Give a verbal example.
- **Quick-fire option (Config B/C):** Skip Slide 8 practice and do one TCF example per department instead (becomes M5).
