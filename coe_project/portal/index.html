The Programme

A structured journey from foundation to automation

The Flow Forward programme is built on 11 self-contained modules grouped across 3 phases. Mix and match for any session length.

🏗️
Phase 1 — Foundation & Ethics
Build the shared vocabulary. Understand what AI is, what Gemini does, and how to use it safely with the Zero Trust guardrails and TCF prompting framework.
M1M2M3M4M5 ~95 min

Phase 2 — Core Tools Deep Dive
Master the four AI tools available in your Vopak account: the Gemini side panel, the Web App for heavy analysis, Gemini on Mobile, and NotebookLM.
M6M7M8M9 ~3.5–6 hrs
Explore modules →
⚡
Phase 3 — Automate & Build
Go from user to builder. Create custom Gems — your AI colleagues — and build personal automation flows with Workspace Studio and Apps Script.
M10M11 ~4 hrs
